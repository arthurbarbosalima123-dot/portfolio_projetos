Projeto: Dashboard Executivo para Monitoramento de Operações

Este projeto tem como objetivo desenvolver um painel gerencial interativo para uma empresa do setor de delivery de comida, utilizando Python, Streamlit e GitHub. A aplicação foi criada para transformar dados operacionais em insights estratégicos, facilitando a tomada de decisão por parte dos gestores.

Criar uma ferramenta prática e visual para monitorar o desempenho operacional, identificar gargalos e apoiar decisões estratégicas com base em dados reais. O projeto demonstra domínio em análise de dados, visualização interativa e deploy de aplicações analíticas.

As ferramentas utilizadas foram:

  • - Python (Pandas, NumPy): para tratamento e análise de dados públicos.
  • - Visualização de dados: gráficos e mapas para facilitar a interpretação dos indicadores.
  • - Hospedagem em nuvem: painel disponível via link público, acessível em qualquer dispositivo.
  • - GitHub: versionamento e colaboração no desenvolvimento.
  • - Streamlit: para construção de uma interface web acessível e interativa.
